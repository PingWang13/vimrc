vimrc
=====

my vim configuration
### How to install

    sudo apt-get install ctags

    git clone https://github.com/PingWang13/vimrc.git   
    
    sudo cp -rf ~/vimrc/vimrc    ~/.vimrc
       
    sudo git clone https://github.com/VundleVim/Vundle.vim      ~/.vim/bundle/vundle
    
open VIM, then run

    :BundleInstall
