my vim configuration
### How to install

    sudo apt-get install ctags

    git clone https://github.com/PingWang13/vimrc.git   
    
    cp ~/vimrc/vimrc ~/.vimrc
       
    sudo git clone https://github.com/gmarik/vundle.git   ~/.vim/bundle/vundle
    
open VIM, then run

    :BundleInstall
